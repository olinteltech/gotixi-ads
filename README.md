# gotixi-ads

This is where we will have code, Claude skills, and creative generation pipelines for generating ads.

## Overview

This repository contains everything needed to build, automate, and scale ad creative generation for Gotixi, including:

- **Code** — scripts, utilities, and services for ad workflows
- **Claude Skills** — custom Claude Code skills for automating ad creation tasks
- **Creative Generation Pipelines** — end-to-end pipelines for generating ad copy, images, and other creative assets
